# 星海资讯网（2007 风格离线新闻网站）

## 运行方法

1. 用 VS Code 打开整个 `shanjian-2007-news` 文件夹。
2. 最简单：直接双击 `index.html`。
3. 推荐：在 VS Code 安装 **Live Server** 扩展，然后右键 `index.html` → **Open with Live Server**。

## 文件说明

- `index.html`：网页整体骨架。
- `styles.css`：2007 年中文门户网站视觉样式。
- `app.js`：新闻数据、页面切换和本地搜索功能。

## 添加新闻

打开 `app.js`，在最上方 `ARTICLES` 数组中照现有格式增加一条对象即可。正文放进 `paragraphs` 数组；程序会自动将它加入首页、分类页和搜索索引。

## 注意

- 网站完全离线运行，不需要安装 Node.js。
- 原始新闻正文按创作者上传文档录入，未进行剧情改写。
- 页面中的“星海资讯网”、备案号、编辑信息和网站公告属于网页框架占位内容，可自行替换。
